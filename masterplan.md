🦉 [Master Plan] Seowl: 잠들지 않는 도시를 위한 심야 라이프 플랫폼

프로젝트 명칭: Seowl (Seoul + Owl)
비전: "잠들지 않는 서울을 즐기는 모든 이를 위한 심야 지도 및 라이프스타일 SNS"
슬로건: Seoul never sleeps, and neither does Seowl.

1. 코어 타겟 (Target Audience)

🚗 Drive-Owls: 밤 드라이브(북악, 한강), 야간 주차, 뷰를 찾는 자차 보유층.

🛴 Active-Owls: 따릉이, 심야 올빼미 버스, 킥보드로 심야 이동을 하는 뚜벅이/라이더.

💻 Work-Owls: 콘센트가 있고 눈치 보지 않을 새벽 24시 카페가 필요한 프리랜서/학생.

🛡️ Safe-Owls: 막차 이후 시간, 밝고 안전한 대기 장소가 필요한 여성/취객 및 야간 유동인구.

2. 4대 핵심 서비스 (Key Features)

Mobility-Mix (모빌리티 가이드): 운전자 모드(주차, 방지턱, 뷰)와 뚜벅이 모드(N버스, 따릉이) 통합 길찾기.

Owl-Eye Filter (심야 특화 필터): 안심 지수(CCTV, 조명), 작업 적합도(콘센트, 소음), 라스트오더 시간 표시.

Proof of Life (실시간 생존 신고): 포털 지도의 거짓 24시간 정보를 대체할, 유저 체크인 기반 '현재 진짜 영업/혼잡 여부' 검증.

Midnight Lounge (심야 커뮤니티): 위치 반경(예: 1km 내) 유저만 접속 가능한 실시간 새벽 핫스팟 채팅.

3. 핵심 엣지 & 아이디어 (Creative Concepts)

뱀파이어 모드 (UX): 낮에는 흑백이나 스텔스 모드로, 저녁 6시 이후 화려한 네온 다크모드로 앱이 활성화되는 희소성 디자인.

새벽 타임어택 (BM/유저 리텐션): 심야 베이커리/카페의 마감 임박 상품을 새벽 2시에 초특가 푸시 알림으로 뿌리는 기능.

안심 거점 (CSR): 크고 밝은 24시 편의점을 'Seowl Safe Haven(야간 안심 정류소)'로 브랜딩하여 안전 가치 창출.

4. 진입 장벽 (Moat) 및 방어 전략

🚨 네이버/카카오 지도가 '심야 필터'를 만든다면?

방어책: Seowl은 목적지를 찾을 때 쓰는 '단순 유틸리티(지도)'가 아니라, 목적지가 없을 때 들어와서 구경하는 '매거진/SNS'로 포지셔닝한다. 거대 포털은 대국민 대상이라 익명 커뮤니티나 취향 큐레이션을 할 수 없다.
🚨 초기 콜드스타트 (아무도 데이터를 안 올려주면?)

방어책: 서울 전역을 런칭하지 않는다. 오직 '홍대 앞 반경 1km'처럼 좁은 구역만 타겟팅하거나, 심야 배달 라이더/택시 기사들을 '데이터 수집 알바(야경수)'로 고용해 초기 영업 중 데이터를 돈으로 채워 넣는다.

5. 단계별 개발 로드맵 (Development Roadmap)

Phase 0. (기획 & 아키텍처): 다크모드 네이티브 UI 기획, 배터리 소모를 고려한 위치 기반(Geofencing) DB 최적화 (PostgreSQL, Redis 구축).

Phase 1. (MVP 런칭): 극소수 지역(강남/홍대 등)의 심야 데이터만 담은 지도, '실시간 체크인(Proof of Life)' 버튼 및 뚜벅이/자차 필터 구현 (클로즈 베타).

Phase 2. (Community): 특정 핫스팟 기반 실시간 익명 채팅 도입, 사용자 등급화(Gamification).

Phase 3. (Scale-up & BM): 'Seowl Pass(심야 할인 구독권)' 결제 연동, 레드불/핫식스 등 브랜드 콜라보 탑재 앱스토어 정식 런칭.

6. 초기 게릴라 마케팅 (Go-To-Market)

대학가 중간고사 야간 기습: 대학가 시험 기간 새벽 2시, Seowl 앱 체크인 시 몬스터 에너지 선착순 50명 무료 제공. (Work-Owls 유입)

차쟁이 성지 벙개: 북악/잠수교의 심야 푸드트럭과 독점 제휴하여 커뮤니티에 좌표를 찍어 오프라인 만남 유도. (Drive-Owls 유입)

빌런 공감형 숏폼: "네이버지도 '영업 중' 믿고 새벽에 나갔다가 헛걸음해 본 사람?"이라는 릴스 광고로 2030 유저 격한 공감대 형성.

야광/축광 포스터 게릴라: 낮에는 안 보이다가, 밤거리에 가로등 빛을 받으면 빛나는 야광 QR 포스터를 홍대/이태원에 도배.

7. 🛠️ AI 에이전트 개발 파이프라인 (Execution System)

agency-agents(페르소나) + hermes-agent(실행엔진) 조합형 1인 개발 스택

Workflow: 본 seowl_master_plan.md 파일을 최상위 폴더에 두고, 아래와 같이 역할별 Agent를 구동하여 개발을 자동화한다.

🤖 Frontend Agent (UI/모바일 빌더): "App.js에 뱀파이어 모드 UI 뼈대 작성해" (도구: File Write / Terminal)

🤖 Backend Agent (DB 최적화): "심야 트래픽 대응을 위해 유저 체크인 데이터를 Redis 캐시에 올리는 API 구축해"

🤖 Growth Agent (마케터): "인터넷 검색 툴을 사용해 '서울 야간 주차 무료 구역' 50곳을 수집해서 JSON으로 뽑아줘" (도구: Web Search)
